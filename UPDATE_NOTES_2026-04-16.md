# 2026-04-16 Web 端升级说明

## 本次目标

- 将原有 Web 前端升级为 `Vue 3 + Vite` 架构
- 增加用户登录、数据库持久化与接口鉴权
- 将功能拆分为多页面，避免单页堆叠
- 恢复并强化摄像头实时检测
- 新增模型性能统计对比页面

## 主要改动

### 1. 后端鉴权与数据库

- 新增 `webapp/auth_storage.py`
  - SQLite 存储：用户、会话、推理审计
  - 自动建表与字段兼容迁移（含 `model_id`）
- 新增 `webapp/auth_service.py`
  - 注册、登录、退出、改密
  - Token 认证
  - 推理行为审计
  - 模型性能聚合统计（含时间字段）

### 2. 后端接口升级

- 更新 `webapp/app.py`
  - 新增认证接口：`/api/auth/*`
  - 原推理与历史接口改为登录后访问
  - 新增模型性能接口：`/api/auth/model-performance`
  - 根路由优先返回 Vue 构建产物

### 3. 前端 Vue 重构

- 新增 `frontend-vue/` 工程
- 路由拆分为多页面：
  - 总览
  - 图片检测
  - 视频检测
  - 摄像头实时检测
  - 历史与审计
  - 模型性能对比
  - 账户设置
- 新增统一布局与侧边导航
- 新增全局模型参数控制

### 4. 摄像头功能恢复

- 独立页面 `CameraView.vue`
- 支持：
  - 打开摄像头
  - 开始/停止实时推理循环
  - 关闭摄像头
  - 实时结果图、目标数、耗时、FPS 展示

### 5. 模型性能页面优化

- 去掉运行次数/FPS/延迟条形图，仅保留对比表格
- 增加时间维度字段：
  - 最近一次时间
  - 最早一次时间
  - 样本时间（最近 5 次）
- 若仅使用一个模型，显示单模型统计提示

### 6. UI 可读性修复

- 修复模型下拉框选项可读性问题（深色背景+亮色文字）

## 运行方式

```bash
cd frontend-vue
npm install
npm run build
cd ..
python run_web.py
```

访问：

```text
http://127.0.0.1:8000
```
